## 4. Brand Voice Recommendation

### Tone: **Calm confidence**

Think Vercel, Linear, Raycast. Not loud. Not salesy. Just... competent.

### Principles:

| Do | Don't |
|----|-------|
| "Deploy in 60 seconds" | "AMAZING AI HOSTING!!!" |
| "We handle infrastructure" | "We're the #1 best platform" |
| "Your agent, always on" | "Revolutionary game-changing AI" |
| Use specifics (60s, 99.9% uptime) | Use superlatives without proof |
| Short sentences. White space. | Walls of text. Feature dumps. |

### Voice characteristics:

- **Minimal** â€” Say less. Mean more. Every word earns its place.
- **Technical but human** â€” Developers are the audience but don't gatekeep with jargon.
- **Confident, not arrogant** â€” "We handle this well" not "Nobody does this better."
- **Action-oriented** â€” CTAs are verbs: Deploy, Connect, Launch. Not "Learn more."

### Color/aesthetic direction:

- Dark mode default (like Vercel, Linear)
- Monospace for code/technical elements
- One accent color (electric blue, soft purple, or amber)
- Generous whitespace
- Subtle gradients, no busy patterns
